CSS Cheat Sheet - Selectors

Basic Selectors

Selector Description
* All elements
div All div elements
div, p All div and paragraph elements
div p All paragraph elements in div elements
div > p All direct descendant paragraphs in div elements
div + p All paragraphs right after div element
div ~ p All paragraph elements that are subsequent siblings of div
.class All elements with specified classname
#id All elements with specified idname
div.class All div elements with specified classname
div#id All div elements with specified idname
